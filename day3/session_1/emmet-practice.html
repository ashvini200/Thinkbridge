Master your VS Code editor skills with these powerful Emmet abbreviations and shortcuts

What is Emmet?

Emmet is a toolkit that greatly improves HTML & CSS workflow. It allows you to write abbreviations that expand into full HTML markup. Originally called Zen Coding, it has become an essential tool for web developers.

Pro Tip

In VS Code, simply type the Emmet abbreviation and press Tab to expand it. Try these examples in your editor! You can also use Ctrl+Shift+P and search for "Emmet" to see all available commands.

1. HTML5 Boilerplate

Create a full HTML5 document structure with just one abbreviation.

Abbreviation

html:5

Expanded Result

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

2. List Generation

Create an unordered list with 5 items using the multiplication operator.

Abbreviation

ul>li*5

Expanded Result

<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>

Rendered Output:

  • List item 1
  • List item 2
  • List item 3
  • List item 4
  • List item 5

3. Grid Structure

Create a container with a row and three columns - perfect for grid layouts.

Abbreviation

div.container>div.row>div.col*3

Expanded Result

<div class="container">
    <div class="row">
        <div class="col"></div>
        <div class="col"></div>
        <div class="col"></div>
    </div>
</div>

Rendered Output:

Column 1
Column 2
Column 3

Emmet Tips & Tricks

  • Use Tab to expand your abbreviations
  • Combine with CSS selectors for complex structures
  • Use $ for numbering (e.g., li.item$*5)
  • Wrap with abbreviation: Select text and use Ctrl+Shift+P â†’ "Emmet: Wrap with Abbreviation"
  • Use {} for adding text inside elements (e.g., div{Hello World})
  • Use ^ to move up one level in the HTML tree
  • Use () for grouping elements together
