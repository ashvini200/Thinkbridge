What is Emmet?

Emmet is a toolkit that greatly improves HTML & CSS workflow. It allows you to write abbreviations that expand into full HTML markup.

// Example: Type "div.container>ul#nav>li.item$*5>a" then press Tab

HTML5 Boilerplate

Create a full HTML5 document structure with one abbreviation:

html:5

Expands to:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

List Generation

Create an unordered list with 5 items:

ul>li*5

Expands to:

<ul> <li></li> <li></li> <li></li> <li></li> <li></li> </ul>

More Emmet Examples

nav>ul>li*3>a

Creates a navigation with 3 list items containing links

div#header+div#main+div#footer

Creates a page structure with header, main and footer divs

table>tr*3>td*4

Creates a table with 3 rows and 4 columns each
